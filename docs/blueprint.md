# **App Name**: Smart Timetable Manager

## Core Features:

- Teacher Profile & Availability Management: Allows administrators to input teacher names, their specific availability (e.g., preferred times, unavailable blocks), and qualifications for teaching specific units.
- Unit Definition: Enables the definition and management of academic units, including prerequisites and unit type (e.g., theory/practical).
- Scheduling Rule Configuration: Allows administrators to configure a set of rules (e.g., teachers cannot teach two classes simultaneously, teachers must be qualified for assigned units) that the timetable generator must adhere to.
- Automated Timetable Generation: An AI-powered tool that automatically generates a conflict-free timetable by matching teachers to units based on their availability, qualifications, and configured scheduling rules.
- Timetable Viewer & Editor: Displays the generated timetable in an interactive weekly or daily view, allowing for manual adjustments to class slots with immediate visual feedback for conflicts.

## Style Guidelines:

- Clean, modern dashboard for timetable management, with a responsive grid-based layout, optimized for desktop use first, with mobile responsiveness. Features calendar-based interfaces for scheduling and clear presentation of data.
- Supports both Dark and Light mode themes. Specific color coding for timetable elements: Theory (Blue), Practical (Orange), and Conflict (Red) for clear visual distinction.
- Subtle animations for smooth transitions and visual feedback for user interactions, especially during manual adjustments in the timetable editor.
- Utilize a legible and modern font family suitable for data-dense displays.
- Employ a consistent set of clean, outlined vector icons for navigation and actions within the timetable system.
